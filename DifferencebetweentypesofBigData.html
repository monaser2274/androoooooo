    • Difference between types of Big Data

    1.Structured data
    It is dependent and less flexible and Matured transaction and various concurrency technique and Structured query allow complex joiningIt is based on the relational database table.
    2.Semi-structured data
    It is more flexible than structured data but less than flexible than unstructured data and The transaction is adapted from DBMS not matured and Queries over anonymous nodes are possible and It is based on RDF and XML.
    3.Unstructured data
    It is flexible in nature and there is an absence of a schema and No transaction management and no concurrency and An only textual query is possible and This is based on character and library data.
